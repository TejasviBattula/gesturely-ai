# Gesturely ✋

A browser-based real-time hand gesture recognition system built using JavaScript and Google MediaPipe.

## 🚀 Features

- Real-time webcam hand tracking
- Gesture recognition using MediaPipe Hands
- Live gesture-to-text conversion
- Text-to-Speech using Web Speech API
- Gesture activity log with timestamps
- Modern responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Google MediaPipe Hands
- Canvas API
- Web Speech API

## 📂 Project Structure

Currently, the application is implemented in a **single HTML file (`index.html`)**, which contains the HTML structure, CSS styling, and JavaScript logic together for simplicity. Future versions will separate the project into dedicated HTML, CSS, and JavaScript files.

## ▶️ How to Run

1. Clone the repository
2. Open `index.html` in a modern web browser
3. Allow camera permissions
4. Start performing hand gestures
